# Security Policy

## Reporte de vulnerabilidades

Si detectas un problema de seguridad, no publiques la vulnerabilidad directamente en abierto antes de comunicarla.

Puedes reportarla a través de:

- Web: [https://disciplinadigital.es](https://disciplinadigital.es)
- Email: `info@disciplinadigital.es`

## Información recomendada en el reporte

- Descripción clara del problema
- Pasos para reproducirlo
- Impacto estimado
- Entorno afectado
- Evidencias técnicas si existen

## Alcance

Esta política aplica al código publicado de Control Horario Digital y a sus componentes propios.

## Aviso

Se agradece la comunicación responsable de vulnerabilidades para poder analizarlas y corregirlas con la máxima rapidez posible.
