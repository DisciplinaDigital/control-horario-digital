# Marcas y signos distintivos

El codigo fuente de Control Horario Digital se distribuye bajo licencia GNU AGPL v3 o posterior.

Esa licencia se aplica al software, pero no concede automaticamente derechos sobre:

- el nombre `Control Horario Digital`
- el nombre `Disciplina Digital`
- logotipos
- identidad visual del sitio web
- dominios, imagen corporativa o material comercial asociado

Salvo autorizacion expresa y por escrito de su titular, no esta permitido:

- usar esos signos distintivos para sugerir afiliacion, patrocinio o respaldo
- redistribuir versiones modificadas usando la misma identidad comercial de forma que pueda generar confusion
- ofrecer servicios o distribuciones de terceros aparentando ser la version oficial del proyecto

Las redistribuciones permitidas por la licencia del software deben preservar con claridad la separacion entre el codigo licenciado y los signos distintivos reservados.
